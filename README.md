# Install Required Tools
1. Make sure you have installed:
2. Flutter SDK – Download here.
3. Android Studio (with Android SDK and emulator).
4. VS Code (optional) with Flutter and Dart plugins.
5. Java JDK (11 or 17) – required for Gradle.

## Verify installation:
flutter doctor

## Set Up Android Emulator or Physical Device

# Run the Project
Open your Flutter project folder in terminal or VS Code.

## Run:
flutter clean
flutter pub get
flutter run

# (Optional) Run via Android Studio 
Open Android Studio.
Open your Flutter project.
Select main.dart
Choose the target device (emulator/phone).
Click Run.
